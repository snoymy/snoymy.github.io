Manga Text Cleaner

This project use OpenCV and Tesseract OCR to detect text box and balloon.

Program diagram


Program diagram


Program process in step


1.Find text balloon by finding circle and squre shape


2.Find word to comfirm the shape that have found is text balloon


3.Fill color only shape that have text in side. (Use pink color to fill for example)


4.Fill with white color version

⦁⦁⦁

The reason why we don’t use only Tesseract OCR to find text because the image is large compare to the texts in page, so Tesseract OCR can not find text well.
